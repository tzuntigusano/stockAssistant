# Capturas para el README

Coloca aquí las imágenes que enlaza el `README.md` principal. Nombres exactos:

| Archivo         | Qué capturar                                                        |
| --------------- | ------------------------------------------------------------------- |
| `analisis.png`  | La ficha de una acción (ej. AAPL): gráfico + sentimiento + breakout. |
| `cartera.png`   | La portada "Mi cartera".                                            |

## Cómo hacerlas

1. Arranca la app (`start.ps1` / `start.sh`) y abre `http://localhost:5173`.
2. Para `analisis.png`: busca una acción cualquiera y captura la ficha.
3. Para `cartera.png`: abre la portada.
4. Guarda los PNG en esta carpeta con esos nombres.

## ⚠️ Privacidad

La portada de **cartera muestra tus posiciones reales** (valores, acciones,
P&L). Para una captura pública usa una **cartera de ejemplo** (o difumina las
cifras): no publiques tus inversiones reales. La ficha de análisis de un valor
no tiene datos personales, así que esa es segura tal cual.
